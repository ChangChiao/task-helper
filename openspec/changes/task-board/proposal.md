## Why

目前市場上缺少一個專注於「短期、即時性生活任務」的媒合平台。現有平台（如外包網站）偏向長期專業工作，無法滿足「幫忙打蟑螂」、「代排隊」、「跑腿買東西」這類短時間、生活化的需求。本專案要建立一個任務看板，讓有需求的人（發案者）快速找到願意幫忙的人（接案者），解決日常生活中的即時困擾。

## What Changes

- 建立全新的任務看板 Web 應用程式
- 支援兩種角色：發案者（發布任務）與接案者（瀏覽並申請任務）
- 發案者可發布帶有固定分類、地點、報酬的任務
- 接案者可瀏覽、搜尋、篩選任務並提出申請
- 發案者從多位申請者中選擇一位接案者
- 內建站內即時訊息系統供雙方溝通
- 任務完成後雙向互評（星級 + 文字）
- 實名認證確保平台信任與安全
- 金流不經平台，由雙方私下處理

## Capabilities

### New Capabilities

- `user-auth`: 使用者註冊、登入、實名認證機制
- `task-management`: 任務發布、編輯、固定分類、地點綁定、狀態生命週期管理
- `task-browsing`: 任務瀏覽、搜尋、篩選功能
- `task-application`: 接案者申請接案、發案者審核並選擇接案者
- `messaging`: 站內即時訊息系統（Socket.io）
- `review-system`: 雙向互評系統（星級評分 + 文字評價）

### Modified Capabilities

（無既有功能，皆為全新建立）

## Impact

- **前端**：建立完整的 Next.js App Router 應用，包含任務看板、個人中心、訊息中心、評價頁面
- **後端**：Next.js API Routes 處理所有業務邏輯
- **資料庫**：PostgreSQL（Neon 免費方案），透過 Prisma ORM 管理 schema
- **即時通訊**：Socket.io 處理站內訊息
- **檔案上傳**：Cloudinary 處理實名認證文件上傳
- **認證**：NextAuth.js 處理登入/註冊流程
- **部署**：Vercel（前端 + API）+ Neon（資料庫）
- **樣式**：Tailwind CSS
